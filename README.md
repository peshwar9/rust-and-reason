# Rust and Reason - LLM Bootcamp

Code examples for the Rust and Reason LLM bootcamp series.

## Running the Examples

### Day 1: Tokenizer

Learn how LLMs process and tokenize text.

```bash
cargo run --bin tokenizer -- What is the capital of France
```

[Read the blog post](https://open.substack.com/pub/rustandreason/p/how-llms-see-text-hint-they-dont)

### Day 2: Bigram Model

Explore the simplest language model using bigrams.

```bash
cargo run --bin bigram -- fox
```

[Read the blog post](https://open.substack.com/pub/rustandreason/p/the-simplest-language-model-bigrams)

### Day 3: Embeddings

Understand how words are represented as vectors.

```bash
cargo run --bin embedding
```

[Read the blog post](https://open.substack.com/pub/rustandreason/p/words-as-vectors-embeddings)
